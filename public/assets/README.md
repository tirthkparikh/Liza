# Assets Folder

## Images (30 romantic images)
Place your romantic images here:
- `images/1.jpg` through `images/30.jpg`
- Wedding photos
- Romantic moments
- Beautiful scenery
- Couple photos

## Videos
Place your romantic videos here:
- `videos/wedding-video.mp4` - Your wedding video
- `videos/romantic-moment-1.mp4` - Romantic moments
- `videos/romantic-moment-2.mp4`
- `videos/background-video.mp4` - Background video for login

## GIFs
Place animated GIFs here:
- `gifs/hearts.gif` - Animated hearts
- `gifs/sparkles.gif` - Sparkle effects
- `gifs/love.gif` - Love animations

## Usage
The app will automatically use these assets. Make sure to:
1. Name your files as referenced in the code
2. Use supported formats: JPG, PNG, MP4, GIF
3. Keep file sizes reasonable for web performance
